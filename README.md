## 🚀 使い方

### 基本操作

1. **トレーニング開始**
   - ページを開くと自動的に新しいワークアウトが開始されます
   - 担当トレーナーを選択（自主トレの場合は「自主」）

2. **種目の追加**
   - **スーパーセット**: `⚡ スーパーセット`ボタンをクリック
   - **通常種目**: `➕ 通常種目`ボタンをクリック

3. **セットの記録**
   - 重量、自力レップ数、補助レップ数を入力
   - 補助が入った場合は自動でオレンジ色に変化
   - RPEは編集可能（クリックして変更）

4. **過去データとの比較**
   - 種目名を入力すると自動で過去データと比較
   - 同じ器具・角度の記録のみ表示

5. **トレーニング完了**
   - `✅ トレーニング完了`ボタンをクリック
   - データがLocalStorageに保存されます

### データ構造

```javascript
{
  id: "entry_001",
  date: "2025-11-15",
  trainer: "yamada",
  exercises: [
    {
      groupType: "superset", // or "normal"
      order: 1,
      exercises: [
        {
          exerciseName: "ベンチプレス",
          equipment: "barbell",
          benchAngle: "flat",
          sets: [
            {
              weight: 100,
              repsUnassisted: 10,
              repsAssisted: 0,
              rpe: 8
            }
          ]
        }
      ]
    }
  ]
}
```
